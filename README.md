# python-csv-data-analysis
Python program for analyzing student performance data from CSV files, including filtering, statistics, and automated reporting.

### Features
- Reads and parses CSV files into structured data
- Computes per-student averages
- Filters students by section and average score range
- Generates summary statistics (mean, standard deviation, min, max, range)
- Writes cleaned and filtered data back to CSV files

### Technologies Used
- Python
- CSV file handling
- Data analysis and statistics
- Dictionaries, lists, and exception handling

### Example Use Cases
- Academic performance analysis
- Data filtering and reporting
- Automated CSV data processing
